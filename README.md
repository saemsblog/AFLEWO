 # AFLEWO Connect

Africa Let's Worship (AFLEWO) is a non-denominational, all-night worship movement that seeks to unite the body of Christ in Africa. AFLEWO Connect is the digital gateway to this continental transformation through worship.

## Vision
One God. One People. One Africa.

## Features
- **Dynamic Chapters**: Explore active chapters across Kenya, Tanzania, and Rwanda.
- **The Archive**: 20 years of worship captured in high-fidelity media.
- **Stories of Glory**: Personal testimonies from the heart of the movement.
- **Prophetic Calendar**: Stay updated with regional auditions and main events.
- **Elastic Navigation**: High-fidelity interface for seamless content discovery.

## Tech Stack
- **Framework**: Next.js 14
- **Styling**: Tailwind CSS & Vanilla CSS
- **Animations**: GSAP & Framer Motion
- **Database**: Supabase
- **Icons**: Material Symbols via Iconify

## Getting Started
1. Install dependencies: `npm install`
2. Run development server: `npm run dev`
3. Build for production: `npm run build`
